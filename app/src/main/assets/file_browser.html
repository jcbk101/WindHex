WindHex Mobile: File Browser

Navigating the file system
File browser
  • Clicking the "left arrow" button on the menu bar navigates backwards through the file browser.
  • The "Options Icon" on the right side of the menu bar will display an options menu.
  • Clicking on any selection within the list with a folder icon will open up that folder.
  • Clicking on any selection within the list with a file icon will select the file to be opened for editing. The file browser will inform you of the files capabilites--can be edited or not.
  • The selections in the option menu are connected to the system settings menu. These options can be set from this option menu, or they can be set from the System Settings Menu.
  • Clicking the system's back button will allow you to exit the file browser, and then return to a previous task.


File Management
When accessing the options menu, you will see "Create New Folder" and "Create New File". Creating either is not permitted on the "File Storage" level. These commands should function properly inside most directories on the device.
