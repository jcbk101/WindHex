WindHex Mobile: Hex Editor

Hex Editing Window

The "Menu (three horizontal lines)" icon displays a list of all the recently loaded files. Selecting a file in that list will open the file in the editor.

The top toolbar contains action buttons which are listed as follows:
  • "Open File" - Opens a file for editing
  • "Save File" - Save a currently opened file
  • "Search" - Search a file for Hex, Text, Table file, or Unicode data
  • "Replace" - Search a file and replace finds using Hex, Text, or Table file data
  • "Undo" - Undo changes in hexadecimal editing
  • "Settings" - Change system settings
  • "More Options" - display options menu


Options Menu
  • "Default Font size" allows font size resetting. The hex window can be scaled up or down using the "Pinch" method.
  • If any table file is loaded, it can be cleared by tapping "Clear Current Table".
  • "Reset Settings" will reset all system settings to default.
  • "About WindHex" shows developer information.
  • "File Info" shows details about the currently opened file.
  • "Help" displays this window.
  • The "Exit" button will exit WindHex.


Lower Menu Bars and Hexadecimal Keyboard

On the bottom of the screen, three different menu bars, and a single hexadecimal keyboard can be utilized.

The default menu bar allows for a few functions to be performed such as:
Default Menu Bar
  • The "Goto" button allows a user to jump to a file location.
  • "Open Table File" will load a table file into slot #1 or #2.
  • "New Table File" will help in the creation of a new table file.
  • "Edit Current Table File" allows editing to the currently loaded table, or a new file can be create.
  • "Swap Table Files" will enable loading a second table file and also switch between the two files.
Hex and Text Window: Long press
If you long press on the hexadecimal or text data areas, a second menu bar will be displayed. Most of these functions are self explanatory.
Should you long press and "Drag the pointer", up, down, left, or right, the displayed hex and text data will be highlighted.

Hex Window: Tap or Touch
Should you tap on the hexadecimal area, a hexadecimal keyboard with four directional arrows will be displayed. The hexadecimal keys will give you the ability to directly input hexadecimal values into the hex display. The arrow keys will move the cursor around the hexadecimal window.

Text Window: Tap or Touch
Should you tap on the text display, a third menu bar will be displayed. This menu enables direct text editing using either ASCII text or a Table File previously loaded. This can be toggled using the "Edit using Table File" switch.

Searching and Replacing Data

These features are very basic and perform similarly to a word processor.
Search
The searching dialog allows for searches using hexadecimal, table file, ASCII, or unicode input types. You have the option to list all the search results in a scrollable list, and use that list to jump to any location that was returned as a result.

Search and Replace
The search and replace dialog DOES NOT allow Unicode input. You have the option to list all the replaced results in a scrollable list and use that list to jump to any location that was returned "replaced" as a result.

Hexadecimal Undo Feature

You will have a maximum of 300 undo-able hex editing actions that can be performed on the file's data.

Navigating Hexadecimal Window

This section covers the scrolling functions for the hex window display.
Scrolling by Pointer
  • To move the hex window line-by-line, simply drag the pointer up or down slowly.
  • Flinging the pointer up or down allows for browsing the file by larger data blocks or pages.


Scrollbar Thumb
On the bottom of the screen, there is a scrollbar that can be used to navigate the file in larger steps or pages. If there is no visible thumb, it means the data just about fits in one window.

File Comparisons

This section will cover the file compare function. Basic functionality is supported for now in order to maintain speed and effciency.
Comparing two files
  • Clicking the "Compare two files" menu selection on the main hex editing screen will open the file browsing window. Once a file is selected, it will be compared against the currently opened file. The top hex display is the current file, and the bottom dislay is the comparing file. Hex differences will be highlighted in RED and YELLOW.
  • Clicking the two arrows will scroll the data either back one page or forward one page.
  • The scrollbar in the center will scroll through pages for faster viewing.
