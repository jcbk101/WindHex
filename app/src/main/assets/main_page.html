WindHex: Home Screen

Home Screen Menu Bar
  • On the menu bar, clicking the "Arrow (Exit)" icon on the left side exits WindHex and returns to the desktop.
  • The "Gear (Settings)" icon on the right side of the menu bar displays the System Settings menu.
  • The "Clock (Recent files)" icon opens up a recent files list. You can reload a file from here instead of searching for the file within the file browser.

Home Screen Buttons
  • Clicking the "Open File" button will display the file broswer.
  • The "About WindHex" button leads to a dialog that displays developer information.
  • The "Help" button will give you access to this help section.
